# Корпус 1 · Этаж 1 — согласование переходных узлов

Статус: расположение переходных узлов подтверждено пользователем 19.06.2026. Рабочая карта и существующая маршрутизация не изменены.

[Открыть нумерованную карту](./b1-floor1-transition-nodes.png)

На плане подтверждены две внутренние лестницы, два наружных выхода и отдельный переход из Корпуса 1 в Корпус 2.

| ID | Визуальное расположение | Подтверждённое назначение |
|---|---|---|
| F1-S01 | Левая внутренняя лестница | Связь с F2-S01 |
| F1-S02 | Правая внутренняя лестница | Связь с F2-S02 |
| F1-X01 | Левый наружный лестничный выход | Наружный выход, не межкорпусный переход |
| F1-X02 | Правый наружный лестничный выход | Наружный выход, не межкорпусный переход |
| F1-B01 | Проём в верхней стене правой части плана | Переход из Корпуса 1 в Корпус 2 через первый этаж |

До получения фактического плана после F1-B01 используется явно обозначенная временная схема Корпуса 2: вход B2-E01, условный коридор и тестовое назначение B2-P01. Временная геометрия не является фактической планировкой.
