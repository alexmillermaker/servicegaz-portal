## Правила использования

- Эти ID относятся к объектам на общем генплане, а не к помещениям внутри корпусов.
- Не переименовывать ID без подтверждения пользователя.
- При подключении общей карты в навигацию использовать этот индекс как источник соответствия.
- Если объект не является корпусом, не добавлять ему этажи и помещения без отдельного подтверждения.
